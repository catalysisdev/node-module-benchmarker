## Development

### Setup

1. Clone the repository: `git clone https://github.com/pepicrft/node-module-benchmarker.git`.
2. Install dependencies: `pnpm install`.
3. Build: `pnpm build`
4. Test: `pnpm test`.

### System requirements

- [NodeJS](https://nodejs.org) 19.1.0
- [PNPM](https://pnpm.io) 7.17.0
